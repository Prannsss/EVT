# **App Name**: Elimar Spring Garden Resort

## Core Features:

- Landing Page: Display a welcoming landing page with a header, sub-heading, and a call-to-action button for booking.
- Top Bar Navigation: Implement a top navigation bar with links to Home, Accommodations, Gallery, and Login pages.
- Signup Page: Provide a user-friendly signup page for new users to create accounts.
- Login Page: Implement a secure login page for registered users to access their accounts.
- Theme Toggle: Add a light and dark mode theme toggle for user preference.

## Style Guidelines:

- Primary color: Soft blue (#A7D1AB) to represent serenity and nature. This color evokes tranquility, mirroring the resort's promise of relaxation within nature.
- Background color: White (#FFFFFF) for a clean and fresh aesthetic.
- Accent color: Light green (#90EE90), approximately 30 degrees to the 'left' of blue on the color wheel, offering a contrasting yet harmonious complement.
- Dark Mode Theme: Implement a dark mode theme with darker shades of blue and gray for the background and text colors for comfortable nighttime viewing.
- Headline font: 'Montserrat' (sans-serif) for headlines and key text to convey modern elegance. Note: currently only Google Fonts are supported.
- Body font: 'Instrument Sans Italic' (sans-serif) for a softer, more approachable feel. Note: currently only Google Fonts are supported.
- Ensure a responsive layout for compatibility across various devices.
- Employ clean, nature-inspired icons for intuitive navigation and visual appeal.
